3. Data Quality Monitoring System


Companies care deeply about:

Missing values

Broken pipelines

Data inconsistencies

The Problem

Data arrives daily from different sources.

But:

Some files are incomplete

Some records are duplicated

Some fields are missing

Your job is to detect these issues automatically.
